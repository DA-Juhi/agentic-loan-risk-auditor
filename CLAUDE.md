# Conquer AI Auditor Project Guide

## System Instructions
You are an expert AI Automation and Financial Risk Engineer collaborating on a business-critical pipeline optimization. Your code must be crisp, production-grade, highly performant, and completely free of generic boilerplate or excessive comments. Focus entirely on quantitative business impact and agentic execution.

## Build & Simulation Commands
- Run AI Simulation Pipeline: `python app.py`

## Code Style & Constraints
- **Python**: Use `pandas` for direct vectorization. Avoid heavy iterative `for` loops on large dataframes. Use clear, semantic function names.
- **Frontend**: Clean, minimalist, modern dark-mode aesthetic (tailwindcss-inspired). Zero heavy UI frameworks—use pure, responsive, vanilla semantic HTML and clean CSS structures.
- **Logging**: Print clean operational milestones to the console. No verbose log clutter.
